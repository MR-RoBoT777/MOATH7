Scanning : الفحص

What is Scanning? It is a set of steps through which you can know all the hosts and the ports, as well as the existing services that are fully functional in the hosts Live hosts, services and ports .. Of course, this is one of the information gathering methods that helps the pen-tester to make a complete profile of the correct Network Map and includes fully functional Targets. What are the expected results after the scanning process ends? 1- Discover all hosts that are already working and reduce the size of the information to real information "Imagine you have 120 hosts, of which 10 are UP & Running." 2- Discover the operating systems in which the hosts and the System Architecture work 3- Discover the services that are already working on all hosts completely 4- Starting to discover some "Vulnerabilities" that some Hosts have What are the tools used in the Scanning stage? There are many tools, but the most important of them is NMAP, and from others there is really no pen tester that can fully practice its tasks. Before we go into the Scanning Tools, we must know the Flags of the TCP Protocol in order to determine the type of Scanning. Of course, we know the difference between TCP and UDP, and because many of the scanning methods depend on the Flags that are present in the TCP Packet because the Flags are simply used to determine the flow of the connection between the source and the destination. TCP Protocol Flags: URG (Urgent) Flags urg sends what means sip the data is sent in a hurry and work on it immediately FIN (Finish) Here we have saved all the data and there is no data You send it again and the next result is the RST for the connection itself. RST (RESET) The connection between the Source and Destination ends, and if in any data, a reconnection must be made again PSH (PUSH) The data here is very important and should be uploaded to the upper layers as soon as possible ACK (Acknowledgment) Confirmation of receipt of data or confirmation that the communication has received what is in this communication. Syn (synchronize) It is to establish the connection from the beginning and the synchronization between the source and the destination
